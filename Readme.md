# Blog API

API RESTful pour la gestion d'articles de blog développée avec Node.js, Express et SQLite.

## Fonctionnalités

- ✅ Création d'articles
- ✅ Consultation de tous les articles avec filtres (catégorie, auteur, date)
- ✅ Consultation d'un article spécifique
- ✅ Recherche d'articles par titre ou contenu
- ✅ Mise à jour d'articles
- ✅ Suppression d'articles
- ✅ Validation des données
- ✅ Documentation Swagger intégrée
- ✅ Protection contre les attaques (rate limiting, helmet, CORS)

## Technologies utilisées

- **Node.js** - Environnement d'exécution
- **Express.js** - Framework web
- **SQLite** - Base de données légère
- **Swagger** - Documentation API
- **Express Validator** - Validation des données
